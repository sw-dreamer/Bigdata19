주요 컬럼 정리
- Survived(목표 변수) : 0(사망),1(생존)
- Pclass(등급) : 1(1등석), 2(2등석), 3(3등석)
- Sex(성별) : male(남자), female(여자)
- Age(나이) : 데이터 타입 float(실수), 연속 데이터 => 범주형 변경 필요(카테고리컬 데이터)
- SibSp (형제/배우자 수) 
- Parch (부모 자식 수)
- Cabin (방이름 (NaN 값 존재))
- Fare(티켓 가격)
- Embarked(승선 항구 ) : C (Cherbourg), S(Southampton), Q(Queenstown)
