
```
import numpy as np
import matplotlib.pyplot as plt
%matplotlib inline

np.random.seed(0)

# y=4x+6을 근사(w1=4,w0=6) 임의의 값을 노이즈를 위해 만듦
X=2*np.random.randn(100,1)
y1=6+4*X
y2=6+4*X+np.random.randn(100,1)

# X,y 데이터 세트 산점도로 시각화
plt.scatter(X,y1)
plt.scatter(X,y2)
plt.show()
```
